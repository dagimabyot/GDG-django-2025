# Todo CLI App (OOP-Based)

## Overview
A simple command-line Todo application built using Python and Object-Oriented Programming (OOP). 
Tasks are saved to a local JSON file (`todos.json`) so that they persist after closing the app.

## Features
- Add new tasks
- View all tasks
- Update tasks (title or completion status)
- Delete tasks
- Persistent storage with JSON

## How to Run
1. Make sure Python 3.x is installed
2. Clone the repository
3. Open terminal and navigate to the project folder
4. Run the app:

```bash
python todo_app.py
