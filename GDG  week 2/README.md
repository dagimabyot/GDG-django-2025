# GDG Week 2 - Python Practice Questions 
 
This folder contains Python practice questions for Week 2 of the GDG Django course, covering basic programming and file handling. 
 
## Questions 
 
1. Discount Check - Ask user for item count and print "Discount applied" if , otherwise "No discount". 
2. Expensive Products - Write a function get_expensive_products(prices) that returns prices 
3. User Activity Log - Log "User logged in" to log.txt, then print the file. 
4. Student Grade System - get_grade(dict, name) with exception handling. 
5/6. Sales File - Read sales.txt, ignore invalid lines, sum valid numbers. 
7. FizzBuzz - Return list with Fizz/Buzz rules up to n. 
8. Move Zeroes - Move all zeros to the end of a list in-place. 
9. Two Sum - Return indices of two numbers that add up to target. 
10. Palindrome Check - Check if integer x reads the same backward. 
11. Square Root - Return integer floor of sqrt without using built-in exponent. 
12. Numbers File - Read numbers.txt, skip invalid, print sum. 
13. Uppercase File - Read file, convert all text to uppercase, handle missing file. 
14. Student Scores Lookup - Ask for name, print score or "Student not found!". 
15. Word Frequency - Count frequency of words in a sentence. 
16. Reverse Dictionary - Swap keys/values, grouping duplicates. 
 
## Usage 
Run each .py file individually using Python 3.
